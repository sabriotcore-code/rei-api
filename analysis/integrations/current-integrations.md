# Current Integrations

## Introduction
The current integrations between systems are documented below.

## Collections Management System
* Customer Information API: Retrieves customer information from the CRM system
* Payment Processing API: Processes customer payments

## Customer Relationship Management System
* Customer Interaction API: Logs customer interactions

## Payment Processing System
* Payment Status API: Retrieves payment status

## Data Flow
The data flow between systems is as follows:
* Collections Management System -> Customer Relationship Management System: Customer information
* Collections Management System -> Payment Processing System: Payment information
* Payment Processing System -> Collections Management System: Payment status
