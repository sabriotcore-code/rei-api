# QUO SMS System Capabilities
The QUO SMS system provides the following capabilities:
* Sending and receiving SMS messages
* Templating for automated message generation
* API endpoints for integration with other systems
* Support for multiple data formats

## API Endpoints
The QUO SMS system provides the following API endpoints:
* `POST /send-sms`: Send an SMS message
* `GET /receive-sms`: Receive an SMS message
* `POST /template`: Create a new template
* `GET /template`: Retrieve a template

## Data Formats
The QUO SMS system supports the following data formats:
* JSON
* XML

## Current Integration Points
The QUO SMS system is currently integrated with the following systems:
* PME Master Sheet (data management)
* Gmail (email workflows)

## SMS Triggers
The QUO SMS system supports the following SMS triggers:
* Automated workflows
* Manual user input

## Templates
The QUO SMS system supports the following template types:
* Pre-built templates
* Custom templates

## Response Handling
The QUO SMS system handles responses in the following ways:
* Automated workflows
* Manual user review

## Workflow Touchpoints
The QUO SMS system has the following workflow touchpoints:
* PME Master Sheet (data management)
* Gmail (email workflows)
