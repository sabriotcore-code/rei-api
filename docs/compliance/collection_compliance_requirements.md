# Collection Compliance Requirements
## Introduction
This document outlines the compliance requirements and audit trail capabilities for the PME Master sheet collection workflows.
## Regulatory Requirements
The collection activities must comply with the following regulatory requirements:
* Data Protection Act
* Information Security Policy
* Records Management Policy
## Audit Log Structures
The audit log will capture the following information:
* Timestamp
* User ID
* Action performed
* Data changed
## Retention Policies
The audit logs will be retained for a minimum of 7 years.
## Compliance Reporting Mechanisms
Compliance reports will be generated quarterly and will include the following information:
* Number of audit log entries
* Number of compliance incidents
* Summary of actions taken
