# Email Template Catalog
## Introduction
This document provides an overview of the email templates used in the collections process. It includes information about the template names, purposes, and variables used.
## Template Inventory
The following templates are used in the collections process:
### 1. Payment Reminder
* Template Name: payment-reminder.html
* Purpose: Send a reminder to customers about overdue payments.
* Variables Used: customer_name, payment_due_date, amount_due
### 2. Payment Confirmation
* Template Name: payment-confirmation.html
* Purpose: Send a confirmation to customers about successful payments.
* Variables Used: customer_name, payment_date, amount_paid
### 3. Collections Notice
* Template Name: collections-notice.txt
* Purpose: Send a notice to customers about collections proceedings.
* Variables Used: customer_name, debt_amount, collections agency
